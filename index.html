AI Insight — Integration Architecture Recommendation
Rather than recreating MemberCare's direct point-to-point integrations (which created DI's current technical debt), the recommended pattern is Dataverse as the integration hub: all systems read/write to Dataverse via APIs, events are propagated via webhooks to an Azure Service Bus topic, and downstream systems subscribe. This event-driven architecture decouples systems, eliminates synchronisation delays, and provides an audit trail for all data changes — addressing the core data quality challenges identified in DI's pre-analysis.
